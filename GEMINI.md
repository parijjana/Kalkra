4.  **Regression Testing:** The entire test suite must be run, and `REGRESSION_SUITE.md` must be updated every time a feature is added or modified to ensure zero regressions.

## Getting Started

### Setup
Run `flutter pub get` in `kalkra_app` and all packages.

### Running
- **Mobile/Web:** `flutter run -t lib/main.dart` in `kalkra_app`.
- **Desktop:** `flutter run -d windows` (or `macos`/`linux`).

### Testing
- **Suite-wide:** Use `mcp_dart_run_tests` on all package roots.
- **Manual:** `flutter test` within specific project directories.
